# NeatFlow Cleaning Solutions

A modern cleaning services website built with Next.js, Tailwind CSS, and shadcn/ui.

## 🚀 Features

- Responsive design
- Services listing
- Careers page with application form
- Clean architecture (MVC-inspired)
- SEO optimized

## 🛠 Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion

## 📦 Getting Started

```bash
npm install
npm run dev
